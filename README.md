# Galaxy Chat App

A simple real-time chat application where users can send messages and photos to each other.

## Features

- Real-time messaging
- Photo sharing
- Voice recording and sending
- Galaxy-themed colors and design
- Logo and startup screen
- Responsive design

## Installation

1. Make sure you have Node.js installed.
2. Clone or download the project.
3. Run `npm install` to install dependencies.
4. Run `npm start` to start the server.
5. Open your browser and go to `http://localhost:3000`.

## Usage

- Type a message and press Send or Enter.
- Click the camera button to select and send a photo.
- Messages and photos are shared in real-time with all connected users.

## Troubleshooting

- If npm install fails, try running PowerShell as Administrator and set execution policy: `Set-ExecutionPolicy RemoteSigned`
- Make sure port 3000 is not in use.

## Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, Socket.io
